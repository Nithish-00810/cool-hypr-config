# cool-hypr-config

My personal, minimal dotfiles for a complete keyboard-driven Linux desktop experience using **Arch Linux** and the **Hyprland** window manager. 

Featured with a cohesive, dark color scheme designed for efficiency and aesthetics on lightweight hardware.

## 🛠️ Components Included

* **Window Manager:** `Hyprland` (Wayland-based tiling window manager)
* **Status Bar:** `Waybar` (Highly customized top bar for system info and workspaces)
* **Application Launcher:** `Rofi` (Quick app launcher and menu system)
* **Notifications:** `Swaync` (Sway Notification Center for desktop alerts)
* **Terminal Emulator:** `Kitty` (Fast, GPU-accelerated terminal)


## 🚀 How to Use

If you want to replicate this setup, clone this repository and copy the configurations to your local system:

```bash
# Clone the repository
git clone [https://github.com/Nithish-00810/cool-hypr-config.git](https://github.com/Nithish-00810/cool-hypr-config.git)

# Copy configurations to your config directory
cp -r cool-hypr-config/* ~/.config/

```
Note: check wether you install waybar, rofi , swaync, kitty,fish
